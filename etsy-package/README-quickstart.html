Thank you for buying Teachable Machine for Micro:bit! You just picked up the friendliest way to wire a Google Teachable Machine model into real hardware — your classes go from webcam, through Chrome, over Bluetooth, to a BBC micro:bit V2 that reacts on its LEDs. No install, no account. This page covers your first three steps.

First 3 Things To Do

1
Train a model. Visit teachablemachine.withgoogle.com, pick Image or Audio, add a few classes (examples: "Wave", "Clap", "Nothing"), click Train, then Export Model and copy the shareable URL.
2
Open the app & load your URL. Double-click index.html in Chrome. Paste the URL in the box, click Load Model, then Start Webcam. Confidence bars light up as the AI sees you.
3
Connect and send. Power the micro:bit V2. Click Connect, pick your board, press TEST to confirm LEDs react. Then click Start Sending. Flip on Talk to hear the computer say what it sees.

Key Features

Teach: Load any public Teachable Machine model URL — Image or Audio

Trigger: Live webcam classification with confidence bars, right in Chrome

Control: Web Bluetooth pairing to micro:bit V2; each class triggers a unique LED reaction

Extras: TEST button (verify pairing), Talking Computer mode (speech synthesis of the top class)

Privacy
