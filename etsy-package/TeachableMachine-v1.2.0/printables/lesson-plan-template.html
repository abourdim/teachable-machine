Sample Lesson: Hot or Not? The Sensor Detective

Ready-to-teach 45-minute example
Hot or Not? The Sensor Detective
45 minutes
Ages 9–14
STEM / Physics / Data Literacy
Learning Objectives
1. Pair a micro:bit V2 over Web Bluetooth.
2. Read temperature, light, and sound sensors.
3. Graph a measurement over time and identify trends.
4. Explain a physical cause for an observed spike.

Playground features used in this lesson

  • BLE Connect Web Bluetooth pairing with auto-reconnect
  • Live Sensors temp, light, sound, accel, classification all at once
  • TensorFlow.js Graph live rolling plot, fullscreen with F
  • Session Recording press R, export as JSON for later analysis
  • Simulate Mode great backup if a board fails mid-class
