5. tools/ — the build pipeline

Every script is a standalone Node ESM file. Each has a docstring at the top. The short version:

The pipeline, in order

node tools/capture-screenshots.mjs          # all screenshots (EN)
node tools/hero-compose.mjs                 # 4 Etsy hero variants
node tools/theme-morph.mjs                  # 6s theme loop
node tools/generate-gifs.mjs                # 4 demo GIFs
node tools/speed-test-clip.mjs              # 3.2s proof clip
node tools/fake-ble-dialog.mjs              # synth Chrome dialog
node tools/photoreal-board.mjs hero         # Three.js board render
node tools/generate-captions.mjs            # A/B caption bank
node tools/generate-accessibility.mjs       # audio desc + Braille
node tools/chatbot-embed.mjs                # "Ask the product" widget
node tools/generate-print.mjs               # poster A3 + flyer A4 (EN)
node tools/generate-social.mjs              # IG/Twitter/LinkedIn PNGs (EN)
node tools/generate-identity.mjs            # business card + email sig (EN)
node tools/generate-datasets.mjs 100        # 100 CSVs (hardware-optional)
node tools/generate-stand-svg.mjs           # lasercut stand SVG+DXF
node tools/generate-video.mjs               # silent 60s base (EN)
node tools/narrate-video.mjs en             # + Zira voice-over
node tools/qr-inject.mjs --inplace          # QR codes on printables

# Then build the ZIP:
node build-package.js

Localization — one command

node tools/build-localized.mjs fr   # runs everything with --lang fr

Other tools

watermark-zip.mjsPer-order watermarked ZIP. Run at Etsy fulfilment: --buyer "Name" --order "12345".
visual-regress.mjsCatches unintended screenshot changes. --baseline to snapshot, default to diff.
build-docs-html.mjsBuilds HTML twins of every .md in etsy-package/, seller-only/, and output/<lang>/. RTL-aware for AR files. Inserts an EN/FR/AR sibling bar on triplet files. Run after editing any markdown.
capture-config.jsonPer-product config: app container, tab selectors, sensor IDs, themes, synthetic state (including per-language strings). Edit this, not the scripts.
hero-specs.jsonA/B hero variants declaration. Add/edit entries here, rerun hero-compose.mjs.
print-specs.jsonPer-language strings for generate-print.mjs (poster + flyer copy).
social-specs.jsonPer-language strings + format list for generate-social.mjs (IG / Twitter / LinkedIn).
captions/video-captions-{en,fr,ar}.srtThe 8-scene SRT scripts. These drive both the burned-in captions and the TTS narration.

6. output/ — everything generated

The full tree was in §2. The short rules:

  • output/en/ — English-language deliverables. Use these for your main Etsy listing. Zira narration.
  • output/fr/ — French-language deliverables. Use these for a French-audience listing / Pinterest / YouTube. Hortense narration.
  • output/ar/ — Arabic-language deliverables with RTL layout. Naayf narration. Full parity with EN + FR.
  • output/shared/ — language-agnostic (mockups, photoreal board, BLE dialog, datasets, lasercut stand, printable renders). One copy for all locales.
  • output/_tmp/ — ffmpeg / Playwright intermediates. Safe to delete. Gitignored.
Which video do I upload to Etsy? EN listing: output/en/etsy-video-v1.mp4 (Zira narration). FR listing: output/fr/etsy-video-v1.mp4 (Hortense narration). AR listing: output/ar/etsy-video-v1.mp4 (Naayf narration, RTL layout). All three already have audio tracks. Etsy plays muted in the grid, users can unmute to hear voice-over.

8. Common tasks — cheat sheet

Regenerate everything from scratch

node tools/build-localized.mjs en     # full EN pipeline
node tools/build-localized.mjs fr     # full FR pipeline
node build-package.js                 # rebuild the buyer ZIP

Change a printable's text

Edit the source HTML at the etsy-package root (e.g. classroom-poster.html), then re-run node build-package.js to re-render the PNG and rebuild the ZIP.

Update the Etsy listing copy

Edit seller-only/ETSY_LISTING.md. The HTML twin updates manually to match. Then copy/paste into Etsy's listing form.

Launch-day sequence

Open MARKETING-GUIDE.html and scroll to "Launch-day sequence" at the bottom. T-2h → T+1w playbook.

Add a new language

  1. Translate tools/captions/video-captions-en.srt to video-captions-xx.srt.
  2. Add xx: entries to per-language fields in tools/capture-config.json (pillText, pills, activityFeed.entries).
  3. Add xx TEMPLATES block to tools/generate-captions.mjs and RULES to tools/chatbot-embed.mjs.
  4. Install a matching OS TTS voice (SAPI / say / espeak-ng).
  5. Run node tools/build-localized.mjs xx.

Ship a per-buyer watermarked ZIP

node tools/watermark-zip.mjs --buyer "Alice Smith" --order "12345"
# Produces output/shared/buyers/TeachableMachine-AS-12345-v1.2.0.zip
