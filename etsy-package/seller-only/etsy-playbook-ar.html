ترتيب الصور السبع للإعلان

تسلسل الصور (Etsy يعرض 1 و 2 بأكبر حجم)

  1. الصورة البطل المقسومة — على اليسار لقطة كاميرا لطفل يؤدي إيماءة؛ على اليمين مصفوفة LED 5×5 للميكروبِت تُظهر السهم المطابق. عنوان بخط Orbitron فوقها.
  2. واجهة لصق الرابط — لقطة لخانة رابط Teachable Machine وزر Load في المتصفح، مع رابط عيّنة teachablemachine.withgoogle.com/models/... ظاهر.
  3. أعمدة التنبؤ الحية — لقطة قريبة أثناء الاستدلال: 4 أعمدة للأصناف (يد-أعلى / يد-يسار / يد-يمين / محايد) مع نسب الثقة، والعلامة الأعلى مميَّزة.
  4. مجموعة أيقونات LED للميكروبِت — شبكة موسومة: 4 أسهم + قلب + وجه سعيد + مربع + معيّن + معيّن صغير، مع اسم كل علامة تحتها.
  5. تحويل النص إلى كلام مفعَّل — لقطة التطبيق مع زر السماعة في وضع التشغيل وفقاعة ترجمة تُظهر «LEFT» منطوقة.
  6. لقطة من جهة المعلم — واجهة تدريب Teachable Machine على teachablemachine.withgoogle.com مع عيّنات كاميرا ملتقطة لكل صنف.
  7. إنفوجرافيك الخصوصية — «الاستدلال يعمل داخل متصفحك بعد التدريب · بلا رفع · بلا حسابات · Chrome / Edge» على تدرج داكن.
