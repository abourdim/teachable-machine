Description de l'annonce

Panneau de contrôle Bluetooth basé navigateur pour le BBC micro:bit V2. Rien à installer. Fonctionne sur Chromebook. Nécessite Chrome ou Edge — pas iPhone / iPad / Safari.

✅ Testé sur Chromebook — appareil cible principal pour les écoles. S'installe comme une PWA, fonctionne hors ligne après le premier chargement.

🎮 Teachable Machine transforme votre micro:bit en laboratoire STEM interactif — rien à installer, aucun compte, aucun cloud. Ouvrez-le dans votre navigateur et connectez-vous.

🔒 Aucun compte. Aucune connexion. Aucune donnée collectée. Sûr pour les élèves de moins de 13 ans — conforme RGPD par conception.

🧪 Essayez maintenant (mode Simulation, sans micro:bit) : https://abourdim.github.io/teachable-machine/#lang=fr

✨ Ce que vous obtenez (téléchargement instantané) : • Une application web complète (HTML, CSS, JS) exécutable localement en Chrome ou Edge • Firmware micro:bit prêt à flasher (makecode.ts) • Guide utilisateur illustré trilingue (EN/FR/AR) avec diagrammes intégrés : — schéma d'architecture • brochage du micro:bit V2 • storyboard du flux firmware — terrain de jeu LED interactif dans le guide (cliquez les cellules, copiez les commandes hex) — dépannage visuel : arbre de décision, sélecteur de symptôme, galerie d'icônes, recette de réinitialisation • Guide de démarrage imprimable (SETUP.md) • 4 thèmes (Stealth, Neon, Arctic, Blaze) • 8 onglets : Matrice LED, Capteurs, Classes, Manette, Graphique, Carte 3D, Banc, Paramètres • 5 modèles 3D interactifs (micro:bit, Buggy, Bras robot, Jeu d'équilibre, Station météo) • Enregistrement de sessions, graphiques en direct, annotations, export JSON • Modes Débutant & Expert — évolue avec l'apprenant

🧒 Idéal pour : salles de classe, école à la maison, clubs périscolaires, fablabs, camps STEM, passionnés de robotique, parents qui initient leurs enfants au code.

⚙️ Nécessite : • BBC micro:bit V2 (vendu séparément) • Navigateur Chrome ou Edge sur ordinateur de bureau, portable ou tablette Android • Bluetooth activé

❌ Ne fonctionne PAS sur : Safari, Firefox, iPhone, iPad. (Limitation de Web Bluetooth — pas de l'application.)

♿ Accessibilité : descriptions audio (voix Hortense fr-FR), transcriptions texte pour lecteurs DAISY, Braille Unicode Grade 1 français. Conforme aux exigences procurement des académies françaises (RGAA 4).

FAQ à ajouter à l'annonce

Q : Dois-je savoir coder ? Non — il suffit de flasher le firmware une fois et de cliquer Connecter.

Q : Est-ce que ça fonctionne sur mon iPad ? Non. Apple n'autorise pas Web Bluetooth dans les navigateurs iOS. Utilisez un ordinateur portable, un Chromebook ou une tablette Android.

Q : Puis-je l'utiliser dans ma classe de 30 élèves ? Oui — un seul enseignant, une seule école, élèves illimités. Pour plusieurs enseignants ou une licence académique, contactez-moi.

Q : Proposez-vous des remboursements ? Comme il s'agit d'un téléchargement instantané, je ne peux pas offrir de remboursement une fois le ZIP livré — vérifiez donc la compatibilité (Chrome ou Edge, ordinateur, BBC micro:bit V2) avant l'achat. Si le produit ne fonctionne pas comme décrit sur un navigateur pris en charge, contactez-moi sous 14 jours et je vous rembourserai personnellement.

Q : Est-ce sûr pour les élèves de moins de 13 ans (RGPD) ? Oui. Tout s'exécute localement dans le navigateur — aucun compte, aucune connexion, aucune collecte de données, aucun cloud, aucune télémétrie. Rien ne quitte l'appareil. Conforme RGPD par conception.

Q : Y aura-t-il des mises à jour ? Oui — mises à jour gratuites dans la ligne v1.x (nouvelles fonctionnalités et corrections de bogues), plus correctifs garantis pendant 12 mois après votre achat. Une future version majeure v2.x serait une annonce distincte avec une remise de mise à niveau pour les acheteurs existants.
