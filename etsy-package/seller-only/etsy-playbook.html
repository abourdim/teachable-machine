Full description

🎮 Teachable Machine for Micro:bit — a browser-based BLE control panel for the webcam + BBC micro:bit V2. Open one HTML file in Chrome or Edge, pair over Bluetooth, and unlock train + predict of hands-on STEM fun — no install, no account, no backend, no tracking. WHAT YOU GET • 🎛 LED matrix + preset icons + text scroller • 📡 Live sensors: temperature, light, sound, ML model, classification, touch • ⚙️ 2 classes (P1 + P2) with sliders, gauges, trim, OFF buttons • 🎮 D-pad Samples for robots and games • 📈 Live Graph (TensorFlow.js) — record, replay, annotate, export PNG + CSV • 🎲 5 interactive 3D models (Three.js): micro:bit, Buggy, Arm, Balance, Weather • 🎨 4 themes (Stealth · Neon · Arctic · Blaze) + 3 languages (EN · FR · AR with full RTL) • 📱 PWA — installable, offline-capable • 👶 Beginner mode + 🧙 Expert mode WHAT'S IN YOUR DOWNLOAD • index.html — the full app • makecode.ts — micro:bit firmware (flash once from MakeCode) • SETUP.md — 5-minute quick-start • docs/ — illustrated guide, cheat sheet, FAQ, Etsy playbook • LICENSE — personal / classroom single-user license REQUIREMENTS • webcam + BBC micro:bit V2 (V1 works but has no sound sensor) • Chrome or Edge on desktop (or Chrome on Android) • Bluetooth enabled • HTTPS or file:// — double-clicking the HTML works WHO IT'S FOR Teachers running STEM workshops · home-schooling families · robotics clubs · makers prototyping BLE projects · anyone curious about the micro:bit. PRIVACY FIRST Nothing leaves your computer. No analytics, no logins, no cloud. All calibrations, recordings, and presets live in your browser's localStorage and on your disk. LIFETIME UPDATES Every version shipped from this listing is yours forever. Re-download the latest ZIP anytime from your Etsy Purchases page. Happy hacking! 🚀🎮🤖
