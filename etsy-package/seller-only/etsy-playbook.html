Full description

🤖 Teachable Machine for Micro:bit — teach a webcam model, trigger a label, control a BBC micro:bit V2. Paste any Google Teachable Machine image-model URL, hit Load, and the browser runs inference locally on TensorFlow.js. Every predicted label is sent as CMD:LABEL,CONFIDENCE over Bluetooth Nordic UART to your micro:bit, which lights a matching icon on its 5×5 LED display. WHAT YOU GET • 🧠 Load ANY Teachable Machine image model by URL — paste → Load → predict • 📷 Live webcam inference — top label + confidence updates in real time • 📡 Web Bluetooth UART bridge to BBC micro:bit V2 (Nordic UART 6e400001) • 🎯 9 built-in label → icon mappings on the micro:bit: LEFT/RIGHT/UP/DOWN → arrow West/East/North/South RED → ♥ Heart · GREEN → :) Happy · BLUE → ■ Square YELLOW → ♦ Diamond · WHITE → · SmallDiamond TEST → "T" · UNKNOWN → "?" · STOP → clear • 🗣 Optional speech synthesis — reads the predicted label aloud (en-US) • 🔄 Front/back camera switch + mirror toggle • 🌐 Works on any laptop or Chromebook with a webcam and Chrome/Edge WHAT'S IN YOUR DOWNLOAD • index.html — the full browser app • makecode.ts — micro:bit V2 firmware (flash once from MakeCode) • SETUP.md — 5-minute quick-start • docs/ — illustrated guide, lesson plan, FAQ, Etsy playbook • LICENSE — personal / classroom single-user license REQUIREMENTS • BBC micro:bit V2 (required — V2-only BLE services) • Webcam-equipped laptop or Chromebook • Chrome or Edge on desktop (or Chrome on Android) • Bluetooth enabled • HTTPS or file:// — double-clicking the HTML works WHO IT'S FOR Teachers running ML / STEM workshops · home-schooling families · robotics clubs · makers prototyping gesture projects · anyone curious about hands-on machine learning. PRIVACY FIRST Training happens on teachablemachine.withgoogle.com — after that, inference runs entirely in your browser. Nothing leaves your computer. No analytics, no logins, no cloud. Webcam frames never hit a server. LIFETIME UPDATES Every version shipped from this listing is yours forever. Re-download the latest ZIP anytime from your Etsy Purchases page. Happy teaching! 🤖📷🚀
