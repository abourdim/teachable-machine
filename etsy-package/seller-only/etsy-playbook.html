Full description

🎓 Teachable Machine for Micro:bit — the easiest way for kids to train an AI and see it control real hardware. Open one HTML file in Chrome, load any Teachable Machine model by URL (image or audio), turn on the webcam, pair a BBC micro:bit V2 over Bluetooth, and every classification lights up an LED icon on the board. No install. No account. No cloud. WHAT YOU GET • 🎓 Load any Teachable Machine model (image or audio) by URL • 📷 Webcam capture + live classification, right in the browser • 📡 Web Bluetooth pairing to a BBC micro:bit V2 • 🟢 TEST button — verify the BLE link without the camera • 🔊 Optional "Talking computer" — speech synthesis says what the AI sees • 🧭 Single-page flow: Train → Load URL → Start Webcam → Connect → Start Sending • 🌐 Runs locally — nothing leaves your computer • 👩‍🏫 Teacher printables: quickstart card, classroom poster, lesson plan, stickers WHAT'S IN YOUR DOWNLOAD • index.html — the full app • makecode.ts — micro:bit firmware (flash once from MakeCode) • SETUP.md — 5-minute quick-start • docs/ — illustrated guide, cheat sheet, FAQ, Etsy playbook • printables/ — A4 / A3 PDFs + editable HTML • LICENSE — personal / classroom single-user license REQUIREMENTS • Computer + Chrome (desktop) or Chrome on Android • Webcam • BBC micro:bit V2 + USB cable or battery pack • Bluetooth enabled WHO IT'S FOR STEM teachers · homeschool families · after-school clubs · maker parents · anyone curious about AI. PRIVACY FIRST The app runs in your browser. No logins. No analytics. No cloud. The model URL you paste goes straight from Chrome to Teachable Machine's CDN — your webcam frames never leave the device. LIFETIME UPDATES Every version shipped from this listing is yours forever.
