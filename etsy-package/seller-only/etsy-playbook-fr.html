Description complète

🎓 Teachable Machine pour Micro:bit — la façon la plus simple pour les enfants d'entraîner une IA et de la voir piloter du vrai matériel. Ouvrez un seul fichier HTML dans Chrome, chargez n'importe quel modèle Teachable Machine par URL (image ou audio), activez la webcam, appairez un BBC micro:bit V2 en Bluetooth, et chaque classification allume une icône LED sur la carte. Sans installation. Sans compte. Sans cloud.
