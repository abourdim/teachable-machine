Description complète

🤖 Teachable Machine pour Micro:bit — entraîne un modèle webcam, déclenche une étiquette, contrôle un BBC micro:bit V2. Colle n'importe quelle URL de modèle image Google Teachable Machine, clique Charger, et le navigateur exécute l'inférence en local avec TensorFlow.js. Chaque étiquette prédite est envoyée en CMD:LABEL,CONFIDENCE via Bluetooth Nordic UART vers ton micro:bit, qui allume l'icône correspondante sur sa matrice 5×5 LED. CE QUE VOUS OBTENEZ • 🧠 Chargez N'IMPORTE QUEL modèle image Teachable Machine par URL — coller → Charger → prédire • 📷 Inférence webcam en direct — étiquette principale + confiance actualisées en temps réel • 📡 Pont Web Bluetooth UART vers BBC micro:bit V2 (Nordic UART 6e400001) • 🎯 9 correspondances étiquette → icône intégrées sur le micro:bit : LEFT/RIGHT/UP/DOWN → flèche Ouest/Est/Nord/Sud RED → ♥ Cœur · GREEN → :) Sourire · BLUE → ■ Carré YELLOW → ♦ Losange · WHITE → · PetitLosange TEST → « T » · UNKNOWN → « ? » · STOP → effacé • 🗣 Synthèse vocale optionnelle — lit l'étiquette prédite à voix haute (en-US) • 🔄 Bascule caméra avant/arrière + miroir • 🌐 Fonctionne sur tout portable ou Chromebook avec webcam et Chrome/Edge CONTENU DU TÉLÉCHARGEMENT • index.html — l'application complète dans le navigateur • makecode.ts — firmware micro:bit V2 (à flasher une fois depuis MakeCode) • SETUP.md — démarrage rapide en 5 minutes • docs/ — guide illustré, fiche pédagogique, FAQ, playbook Etsy • LICENSE — licence mono-utilisateur personnelle / classe PRÉREQUIS • BBC micro:bit V2 (requis — services BLE V2 uniquement) • Ordinateur portable ou Chromebook avec webcam • Chrome ou Edge sur ordinateur (ou Chrome sur Android) • Bluetooth activé • HTTPS ou file:// — un double-clic sur le HTML suffit POUR QUI Enseignants animant des ateliers ML / STEM · familles en école à la maison · clubs de robotique · makers prototypant des projets gestuels · toute personne curieuse de machine learning concret. CONFIDENTIALITÉ D'ABORD L'entraînement se passe sur teachablemachine.withgoogle.com — ensuite, l'inférence tourne entièrement dans votre navigateur. Rien ne quitte votre ordinateur. Pas d'analytique, pas de connexion, pas de cloud. Les images de la webcam ne partent jamais vers un serveur. MISES À JOUR À VIE Chaque version publiée depuis cette annonce est à vous pour toujours.
