Description complète

🎮 Teachable Machine for Micro:bit — un panneau de contrôle BLE dans le navigateur pour le webcam + BBC micro:bit V2. Ouvrez un fichier HTML dans Chrome ou Edge, appairez en Bluetooth, et débloquez un outil STEM interactif — sans installation, sans compte, sans serveur, sans tracking. CE QUE VOUS OBTENEZ • 🎛 Matrice LED + icônes prédéfinies + texte défilant • 📡 Ensemble de signaux et contrôles en direct (voir la description) • ⚙️ 2 classes (P1 + P2) avec curseurs, jauges, trim, boutons OFF • 🎮 Manette D-pad pour robots et jeux • 📈 Export de session — rejouer, annoter, exporter • 🎲 5 (Three.js) : micro:bit, Robot, Bras, Équilibre, Météo • 🎨 4 thèmes (Stealth · Neon · Arctic · Blaze) + 3 langues (EN · FR · AR avec RTL complet) • 📱 PWA — installable, utilisable hors ligne • 👶 Mode Débutant + 🧙 Mode Expert CONTENU DU TÉLÉCHARGEMENT • index.html — l'application complète • makecode.ts — firmware micro:bit (à flasher une fois depuis MakeCode) • SETUP.md — démarrage rapide en 5 minutes • docs/ — guide illustré, aide-mémoire, FAQ, playbook Etsy • LICENSE — licence mono-utilisateur personnelle / classe PRÉREQUIS • webcam + BBC micro:bit V2 (V1 fonctionne mais sans capteur de son) • Chrome ou Edge sur ordinateur (ou Chrome sur Android) • Bluetooth activé • HTTPS ou file:// — un double-clic sur le HTML suffit POUR QUI Enseignants animant des ateliers STEM · familles en école à la maison · clubs de robotique · makers prototypant des projets BLE · toute personne curieuse du micro:bit. CONFIDENTIALITÉ D'ABORD Rien ne quitte votre ordinateur. Pas d'analytique, pas de connexion, pas de cloud. Tous les calibrages, enregistrements et préréglages restent dans le localStorage du navigateur et sur votre disque. MISES À JOUR À VIE Chaque version publiée depuis cette annonce est à vous pour toujours. Retéléchargez le dernier ZIP à tout moment depuis votre page Achats Etsy. Bon hacking ! 🚀🎮🤖
