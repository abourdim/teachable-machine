Pre-flight (90s)

  • Teachable Machine image model trained (2–3 classes)
  • Model URL copied from Export → Upload
  • Chrome (or Edge) open, index.html loaded
  • Webcam permission granted — live preview visible
  • BBC micro:bit V2 powered, Bluetooth pairing ready
  • Bluetooth chip shows green after connect
  • Language matches voiceover (EN / FR / AR)
  • Speech synthesis tested once (optional)
  • Props for the trained gestures ready on desk
  • Phone in 9:16 on tripod, front-lit
  • Desk cleared — laptop, webcam, micro:bit only
  • Script read once, under 55s
  • Phone silenced, kids/dogs warned

During recording

The 60-second arc — Script A

TimeSayDo / Click
0–3s "Teach your micro:bit to see — in the browser, in 60 seconds." Close-up webcam + micro:bit V2 side by side. Hand holds the board.
3–10s "Train a model on Teachable Machine, paste the URL, done." B-roll of teachablemachine.withgoogle.com → Export → Copy URL.
10–20s "Paste the model URL — the webcam starts classifying instantly." Paste URL in app. Live webcam preview + confidence bars on screen.
20–30s "One click pairs the micro:bit over Bluetooth." Click Connect micro:bit → Chrome picker → V2 selected → chip green.
30–40s "Show a gesture — the micro:bit LEDs react live." Perform trained gesture. Top class highlights. Real board shows matching icon.
40–50s "TEST button, optional voice feedback, three languages, no install." Click TEST. Toggle Speech. Language EN → FR → AR swap.
50–55s "Link below — lifetime updates, teachers and kids welcome." Face-cam. Eye contact with lens.
55–60s "Teach. Trigger. Control." Hold still. Logo + handle freeze. Loop frame.

After Stop

  • Watch once, eyes off script
  • Trim to 55s (iMovie / CapCut / DaVinci)
  • Burn in captions — centered below subject
  • Set first frame = strong still visual
  • Export MP4/H.264, < 100 MB
  • Upload · verify autoplay · phone + sound off
