Sample Lesson: Teach a Computer to Read Hand Signs

Ready-to-teach 45-minute example
Teach a Computer to Read Hand Signs
45 minutes
Ages 9–14
AI Literacy / Computer Science / STEM
Learning Objectives
1. Train a Teachable Machine image model with 3 hand-sign classes.
2. Explain what "class" and "confidence" mean in machine learning.
3. Pair a BBC micro:bit V2 via Web Bluetooth.
4. Connect the AI to hardware so each hand sign triggers a unique LED pattern.

Features used in this lesson

  • Load Model Paste a Teachable Machine URL, fetch the AI into the browser
  • Start Webcam Live classification with confidence bars
  • Connect + TEST Web Bluetooth pairing to micro:bit V2, test LED command
  • Start Sending Push the top class every frame — the board reacts
  • Talk Speech synthesis: the computer announces what it sees
