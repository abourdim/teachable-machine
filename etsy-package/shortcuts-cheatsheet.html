Load ModelPaste a Teachable Machine URL, fetch the AI
Start WebcamGrant camera access, begin live classification

TESTSend a single test command — verify pairing
Start SendingPush the top class to the micro:bit each frame
Talk On/OffToggle speech synthesis — the computer speaks the class
Heads-up
No keyboard shortcuts. Everything is button-driven — safe for kids and touchscreens.
Bluetooth Connection States
Disconnected — click Connect to pair
Pairing — choose "BBC micro:bit [xxxxx]" in the browser prompt
Connected — ready to send classifications
Sending — AI results streaming to the board
Error / Lost — power-cycle the micro:bit, pair again
Recommended Flow
1. Load Model → 2. Start Webcam → 3. Connect → 4. TEST → 5. Start Sending
Once connected, you can TEST any time without the webcam running.
What The micro:bit Does
Each class = one LED reaction. The board shows a distinct icon / arrow / face per class on its 5×5 LEDs.
TEST sends a demo icon so you can confirm pairing before training anything.
Talking Computer
Talk On — browser speech synthesis announces the top class ("I see a wave!").
Talk Off — silent; LEDs still react on the board.
Pro Tips
Use Chrome on a computer. Web Bluetooth is not supported in Safari or on iPhone. Good lighting → better AI accuracy. Train more examples per class → fewer mis-fires.
