Teachable Machine for Micro:bit

Train an AI. Point a webcam. Make a micro:bit react.
Teach • Trigger • Control • Chrome + webcam + BBC micro:bit V2
1
🧠

Train a Model

Visit teachablemachine.withgoogle.com, pick Image or Audio, add your classes, click Export Model and copy the shareable URL.

2
🔗

Load the URL

Open the app in Chrome. Paste your model URL into the box and click Load Model. Class names appear on screen.

3
🎥

Start Webcam

Click Start Webcam and allow camera access. Live confidence bars appear for every class the AI recognises.

4
📡

Connect via Bluetooth

Power the micro:bit V2, click Connect, pick BBC micro:bit [xxxxx]. Press TEST to check the LEDs respond.

5
🚀

Start Sending — Make It React

Click Start Sending. The top class gets pushed to the micro:bit over Bluetooth — LEDs show a matching icon for each class. Flip on Talk for the "talking computer" mode: speech synthesis announces what the AI sees.
