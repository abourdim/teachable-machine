Long paragraph

Teachable Machine is a one-tab control panel for the BBC micro:bit V2 — the tiny classroom microcontroller from the Micro:bit Educational Foundation. Traditional micro:bit workflows require an IDE install, a code flash for every experiment, and a teacher fluent in JavaScript or Python. Playground collapses that workflow: the firmware flashes once, then the rest lives in a single browser tab. Web Bluetooth pairs the board in three seconds; sensors stream live; kids drive motors, paint LEDs, and watch a photoreal 3D mirror of the board tilt with every physical movement. Built entirely client-side — no cloud, no accounts, no telemetry — it meets COPPA, GDPR, and typical school-district privacy-vendor reviews out of the box. The illustrated guide ships trilingual (English, French, Arabic) and includes an accessibility pack with audio descriptions and Braille transcripts, making it one of the few STEM tools on Etsy designed for special-education procurement compliance.

Paragraphe long

Teachable Machine est un panneau de contrôle mono-onglet pour le BBC micro:bit V2 — le microcontrôleur pédagogique de la Micro:bit Educational Foundation. Les flux de travail classiques avec le micro:bit imposent l'installation d'un IDE, un flash de code pour chaque expérience et un enseignant à l'aise en JavaScript ou Python. Playground aplatit ce parcours : le firmware se flashe une fois, puis tout vit dans un onglet de navigateur. Web Bluetooth appaire la carte en trois secondes ; les capteurs diffusent en direct ; les enfants pilotent des moteurs, peignent les LEDs, et regardent un miroir 3D photoréaliste de la carte s'incliner à chaque mouvement physique. Entièrement côté client — pas de cloud, pas de compte, pas de télémétrie — il respecte le RGPD, la conformité COPPA et les revues de confidentialité typiques d'une académie dès le départ. Le guide illustré est trilingue (anglais, français, arabe) et inclut un pack accessibilité avec descriptions audio et transcriptions Braille, ce qui en fait l'un des rares outils STEM sur Etsy conçu pour la conformité aux marchés publics de l'enseignement adapté.
