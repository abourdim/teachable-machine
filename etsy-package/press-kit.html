Short paragraph

Teachable Machine for Micro:bit turns any webcam into a classroom ML demo. Train a model on teachablemachine.withgoogle.com, copy the sharable URL, paste it in the browser — TensorFlow.js loads the model locally and starts classifying the webcam. Show a gesture; the label and confidence appear. Long paragraph

Teachable Machine for Micro:bit is a browser-based ML classifier that bridges Google's Teachable Machine to a BBC micro:bit V2. Classic classroom ML workflows either require a Python backend or never leave a dashboard. This tool flattens that journey: train an image model on teachablemachine.withgoogle.com (Google's free browser-based ML trainer), copy the sharable URL, paste it into the app. TensorFlow.js downloads the model, initializes the webcam, and starts classifying locally — no cloud calls after load. Each class maps to one of nine LED icons: heart, arrow, smile, tick, cross, musical note, house, yes, no — swappable at will. Predictions stream as CMD:LABEL,CONFIDENCE over Nordic UART to a BBC micro:bit V2 paired via Web Bluetooth, and the winning label lights up its icon on the 5×5 matrix. The open protocol means any MakeCode firmware can consume the stream — fork the .hex, change the LEDs, build your own game. Fully client-side after the model downloads — no account, no telemetry, COPPA/GDPR-friendly. Trilingual UI (EN/FR/AR with RTL). One of the few Etsy tools that connects real web-trained ML to real hardware output, in a classroom-safe way.
